Getting Started with Flask: A Beginner's Guide

October 15, 2024

üìñ 5 min read

As a Python backend developer, I've found Flask to be an incredibly powerful yet simple framework for building web applications. In this post, I'll share my journey of getting started with Flask and provide a step-by-step guide for beginners.

What Makes Flask Special?

Flask is a micro web framework written in Python. Unlike heavier frameworks like Django, Flask gives you the freedom to choose your own tools for databases, authentication, and other components. This flexibility makes it perfect for small to medium-sized applications and APIs.

Setting Up Your Environment

Before we dive in, ensure you have Python installed (version 3.6 or higher is recommended). Then, create a virtual environment and install Flask: 

python -m venv flask_env
flask_env\Scripts\activate  # On Windows
pip install flask

Building Your First Flask Application

Let's create a simple "Hello World" app. Create a file named app.py: 

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return 'Hello, Flask World!'

if __name__ == '__main__':
    app.run(debug=True)

Run this with python app.py and visit http://127.0.0.1:5000/ to see your first Flask app in action!

Understanding Routes

Routes are the URLs of your application. Here's how to add more routes: 

@app.route('/about')
def about():
    return 'Welcome to my about page!'

@app.route('/user/')
def show_user_profile(username):
    return f'User: {username}'

The is a variable that captures part of the URL.

Working with Templates

For dynamic content, Flask uses Jinja2 templating. Create a templates folder and add HTML files. Here's a simple template: 




    My Flask App


    
{{ title }}

    
{{ content }}

Then in your route:

from flask import render_template

@app.route('/')
def home():
    return render_template('index.html', title='Home', content='Welcome!')

Next Steps

Flask is just the beginning. As you progress, explore Flask extensions for databases (like Flask-SQLAlchemy), user authentication, and API development. The official Flask documentation is an excellent resource.

Remember, the best way to learn is by building. Start with small projects and gradually add complexity. Happy coding!
