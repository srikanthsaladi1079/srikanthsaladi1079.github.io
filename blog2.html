Building a Simple Expense Tracker with Python and Flask

üìñ 8 min read

As a Python backend developer, I've built several expense trackers for personal and client projects. In this post, I'll guide you through creating a simple yet functional expense tracker using Flask and SQLAlchemy. This project is perfect for learning full-stack development and understanding how to integrate databases with web applications.

Why Build an Expense Tracker?

Expense trackers are practical applications that teach core concepts like CRUD operations, database relationships, and user input handling. They also demonstrate how to present data in an organized, readable format.

Project Structure and Setup

Let's start by setting up our project structure. Create a new directory and initialize a virtual environment:

cd expense_tracker
python -m venv venv

Defining the Database Model

We'll use SQLAlchemy to define our Expense model. Create a models.py file: 

from flask_sqlalchemy import SQLAlchemy
from datetime import datetime

db = SQLAlchemy()

class Expense(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    description = db.Column(db.String(200), nullable=False)
    amount = db.Column(db.Float, nullable=False)
    category = db.Column(db.String(50), nullable=False)
    date_created = db.Column(db.DateTime, default=datetime.utcnow)

    def __repr__(self):
        return f''

Flask Application and Routes

Now let's create our main Flask app. Create app.py: 

from flask import Flask, render_template, request, redirect, url_for
from models import db, Expense

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///expenses.db'
app.config['SECRET_KEY'] = 'your-secret-key'
db.init_app(app)

@app.route('/')
def index():
    expenses = Expense.query.order_by(Expense.date_created.desc()).all()
    return render_template('index.html', expenses=expenses)

@app.route('/add', methods=['GET', 'POST'])
def add_expense():
    if request.method == 'POST':
        description = request.form['description']
        amount = float(request.form['amount'])
        category = request.form['category']
        
        new_expense = Expense(description=description, amount=amount, category=category)
        db.session.add(new_expense)
        db.session.commit()
        
        return redirect(url_for('index'))
    
    return render_template('add_expense.html')

if __name__ == '__main__':
    with app.app_context():
        db.create_all()
    app.run(debug=True)
